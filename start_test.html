开始检测

规则说明

在开始之前一定要先阅读以下说明和规则。
1. 如果没有在规定时间内完成或者中途退出，您的检测卷将视为无效；
2. 每张检测卷的首次检测非常重要：获得的成绩会积累所在教材的总分、如果获得满分会进入满分榜；
3. 只有学生的检测会被系统记录，但如果检测卷不是自己所在年级的，系统不会记录；
4. 答题过程中不要使用浏览器的前进和后退，请使用“上一题”、“下一题”进行检查和修改；
5. 英语如果是听力检测，请在开始之前请打开声音、准备好耳机。

如果您是第一次参与检测，建议先完成 体验检测卷 ，体验一下检测过程和规则。
